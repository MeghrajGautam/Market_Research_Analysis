# Amazon vs Flipkart – Excel Data Analysis Project

## Project Overview
This project compares two major Indian e-commerce platforms: **Amazon** and **Flipkart** using Microsoft Excel.

The objective of this project is to demonstrate **basic data analysis and visualization skills** using Excel.

---

## Dataset
The dataset contains comparison factors between Amazon and Flipkart:

- Pricing
- Delivery
- Customer Reviews
- Features

Each platform is given a **rating from 1–5** for analysis.

---

## Tools Used
- Microsoft Excel
- Charts (Bar Chart)
- Basic Data Analysis
- Average Calculation
- Data Comparison

---

## File Structure

**amazon_flipkart_advanced_analysis.xlsx**

Sheets included:

### 1. Comparison Sheet
Contains detailed comparison of Amazon and Flipkart with ratings.

### 2. Chart Visualization
A bar chart that compares ratings of both platforms.

### 3. Summary Sheet
Shows average rating of Amazon and Flipkart.

### 4. Insights Sheet
Business insights derived from the comparison.

---

## Key Insights

- **Flipkart** performs better in **pricing and discounts**.
- **Amazon** performs better in **delivery speed and logistics**.
- Amazon offers more **features such as Prime membership and international products**.
- Flipkart is strong in **electronics and smartphone deals**.

### Final Conclusion
Amazon leads in **service quality and delivery**, while Flipkart leads in **pricing and deals**.
